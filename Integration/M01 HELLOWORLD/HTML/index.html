Le développement

Le Javascript:

Javascript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fontion de certains paramètres.

Le PHP :

Le PHP est un langage de programation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations (heure,nom,formulaire rempli par l'utilisateur etc).

MySQL:

MySQL est un système de gestion de base de données, il permet d'enregistrer des informations remplie par l'utilisateur (exemple:formulaire de création de compte) pour être ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe.)
